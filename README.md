# Galactic Shooter (GameMaker Studio 2)

**Galactic Shooter** is a top-down space shooter built in **GameMaker Studio 2**.  
You pilot a ship using your mouse, survive fast enemy spawns, dodge bullets and asteroids,
collect weapon upgrades, and aim for a high score across multiple stages.

> “The Galactic Empire has invaded your Home World… Destroy them all before they destroy YOU.”

---

## Features

- **Mouse-controlled ship movement**
- **Enemy ships + enemy bullet patterns**
- **Asteroids** as additional hazards
- **Weapon upgrade pickup** (power-up)
- **Score system** (`global.playerScore`)
- **High-score leaderboard** (Game Over screen)
- **Multi-room progression** + **boss stage**
- UI screens: **Start Screen**, **Next Level**, **Game Over**, **End Game**

---

## Controls

- **Mouse** — Move your ship  
- **N** — Next level (on “Congratulations / Next Room” screen)  
- **R** — Retry (on Game Over screen)  
- **Q** — Quit (on Next Room / Game Over screens)

---

## How to Run

1. Install **GameMaker Studio 2** (GMS2).
2. Open one of the project files (`.yyp`) in GameMaker:
   - Main version:  
     `Galactic Shooter/Galactic Shooter.yyp`
3. Click **Run** inside GameMaker Studio to play.
